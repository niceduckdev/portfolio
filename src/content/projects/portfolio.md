This portfolio was built with Astro and styled using Tailwind CSS, featuring the Jellybean color scheme. The content is typeset in the Nunito typeface.

I use Astro Content Collections to be able to write my projects and articles in Markdown and show them on this Astro website.

The whole website is hosted on Github and deployed via Cloudflare Pages.
